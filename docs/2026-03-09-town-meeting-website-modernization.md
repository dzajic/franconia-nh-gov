## Addendum: Optional Use of GitHub for Hosting & Management

As an option (not a requirement), the town could choose to use GitHub as part of the website setup.

**Potential benefits:**

- Full change history (who changed what, and when)
- Fine-grained permissions (no shared admin passwords)
- Easy rollback if something goes wrong
- No vendor lock-in; content remains simple files (Markdown + images)
- Free or very low-cost hosting via GitHub Pages

**Important note:**

GitHub is simply a management and hosting tool.  
The website itself remains standard HTML files, and could be moved elsewhere at any time if the town prefers.
